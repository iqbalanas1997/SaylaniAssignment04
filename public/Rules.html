IT SOLUTION DATA ENTRY RULES & REGULATIONS

____ ____

  • Do not use below symbols any where while typing.
    Single Quotes (‘) e.g. Boy’s (Wrong) Boys (Right).
    Double Quotes (“) e.g. “America” (Wrong) America (Right).
    Question Mark (?) e.g. How are you? (Wrong) How are you (Right)
  • Give one space before & after hyphen (-) in any condition and if hyphen (-) shows in the start or the end will not be mentioned.
  • Only in address Column anywhere after the full stop ( . ) use double space (^^) Not in the end.(^ sign represents Space)
  • After comma (,) put double-space (^^) wherever it appears except:
    Yearly Revenue
    No. Of Employees
    Yearly Expense
  • Form number must be mentioned, as it shows and it is case sensitive,FORM NO. should be mention as same, no change or space will be applied.
  • Always prefix https:// in website field before www. And if www. is not mentioned, you have to mention it
  • Leave no space in Form Number, Website & Email fields anywhere. Whenever (&) it appears in Form Number, Website and Email Field, do not write it in full form.
    e.g. https://www.a&z.com.
  • Remain 27 field will be types as that 1st letter must be in capital and other alphabets in small and if all alphabets written in capital then all must be typed in capital.
  • Wherever any word has 2 capital Alphabets in it, you will follow the same letter case above 1st initial capital and other in small.
  • Whenever & appears in Form Number, Website and in email field, do not write it in full form.
    e.g. https://www.a&z.com
  • Extract fax number, Contact number & Landmark from address & should be filled in their particular field, Do not Type in the Address column.
  • Type full form of date as shown on image data as shown e.g. if it is given as 04/07/1776 write it the down in such a manner 4th July, (^^) 1776).
  • Whatever is not given in the image data, please type Not Mentioned in those Particular fields. Please do not re-correct the spelling mistake if it is mentioned not correct.
  • Use the name of Currency instead of symbols (e.g. if it is given as $ write it in full form
    for example $ - Dollar, £ - Pound, € - Euro, ₫ - Dong
  • Zip Code / Postal Code Rules

    • If word P.O Box mentions with any digit you will write the same number in both places in Address and ZipCode Column.

    If word P.O Box is not mentioned then any 4 to 6 digit number you will consider as Zip-Code and write in both places in address and Zip-Code column.
